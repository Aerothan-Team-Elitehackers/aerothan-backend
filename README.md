# Backend

## Tech Stack

- Node.js
- Express.js
- Supabase
- Postgres
- Netlify
- Serverless Functions

## To run the backend

```bash
npm run start
```

NOTE: The NodeJS version should be `16.20.0`

## To run the backend tests

- Go to URL: [Backend-Home](http://localhost:9000/.netlify/functions/api)

## Netlify Deployment
